# Dependencias de Python para el Proyecto de Sistemas Operativos

# Este proyecto, en su estado actual, utiliza únicamente librerías
# que forman parte de la biblioteca estándar de Python.
# No se requieren instalaciones adicionales mediante pip si tienes
# una instalación de Python 3.x razonablemente moderna (ej. Python 3.7+).

# Librerías estándar utilizadas (implícitamente, no necesitan listarse para instalación):
# - socket (para comunicación de red)
# - threading (para concurrencia basada en hilos)
# - json (para serialización/deserialización de datos)
# - os (para interacción con el sistema operativo, ej. manejo de archivos/directorios)
# - collections (para estructuras de datos como deque)
# - concurrent.futures (para ThreadPoolExecutor y ProcessPoolExecutor)
# - time (para pausas y medición de tiempo)
# - re (para expresiones regulares)
# - math (para operaciones matemáticas como math.ceil)
# - sys (para sys.exit, sys.stdout.flush)
# - tkinter (para la interfaz gráfica de usuario, incluyendo ttk, scrolledtext, etc.)
# - queue (para comunicación segura entre hilos)
# - csv (para leer/escribir archivos CSV)
